Welcome To RTCScope Dashboard

 

 

RTCScope.js is a Javascript library that can extract the WebRTC stats from a WebRTC based applications. This dashboard visaulizes the extracted WebRTC stats in graphs

 

This dashboard provides separate options to visualize stats in Realtime for the Ongoing calls as well as adhoc graphs for finished graphs also

 

For simple testing, a Demo page is also included to simualte a simple WebRTC call from your Browser
